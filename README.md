🏍️ Catálogo de Recorridos en Cuatrimoto

Aplicación web desarrollada con Vue.js que presenta un catálogo interactivo de recorridos en cuatrimoto. El proyecto está enfocado en ofrecer una experiencia visual atractiva mediante videos, imágenes y descripciones detalladas de cada ruta.

🚀 Características
📸 Galería de imágenes por recorrido
🎥 Videos integrados para una mejor experiencia visual
📍 Ubicación del lugar de cada recorrido
📝 Descripción detallada de cada ruta
🎨 Interfaz moderna y responsiva
⚡ Aplicación 100% frontend (sin backend)
🛠️ Tecnologías utilizadas
Vue.js
HTML5
CSS3
JavaScript (ES6+)
📂 Estructura del proyecto
/src
  /assets        # Imágenes, videos y recursos estáticos
  /components    # Componentes reutilizables
  /views         # Vistas principales
  /router        # Configuración de rutas
  App.vue        # Componente raíz
  main.js        # Punto de entrada
📦 Instalación y uso
Clona el repositorio:
git clone https://github.com/tu-usuario/tu-repositorio.git
Entra al directorio del proyecto:
cd tu-repositorio
Instala las dependencias:
npm install
Ejecuta el servidor de desarrollo:
npm run dev
Abre en tu navegador:
http://localhost:5173
📸 Funcionalidades principales
🗺️ Catálogo de recorridos

Cada recorrido incluye:

Nombre del lugar
Imágenes destacadas
Video del recorrido
Descripción detallada
Ubicación
🎬 Experiencia multimedia

El usuario puede visualizar contenido dinámico para conocer mejor cada ruta antes de elegirla.

⚠️ Limitaciones
❌ No cuenta con backend
❌ No hay sistema de autenticación
❌ Los datos son estáticos o simulados
